Parkour Game

A fast-paced first-person parkour game built with Unreal Engine 5, focused on fluid movement, precision platforming, and immersive level design. Master advanced movement mechanics, overcome challenging obstacles, and race through beautifully crafted environments.

Features

- 🏃 Smooth first-person movement
- 🦘 Sprint, Jump, Vault & Wall Run
- 🧗 Ledge Grab & Climbing
- 🎯 Precision platforming challenges
- 🌍 High-quality environments
- 🎮 Responsive controls using Unreal Engine Enhanced Input
- ⚡ Optimized gameplay with Unreal Engine 5

Built With

- Unreal Engine 5
- C++
- Blueprint System
- Enhanced Input System
- Niagara VFX
- Lumen Global Illumination
- Nanite Virtualized Geometry

Installation

1. Clone the repository.
2. Open the ".uproject" file using Unreal Engine 5.
3. Generate Visual Studio project files (if using C++).
4. Build the project.
5. Press Play inside the editor.
